# Introducción a Media Queries 

![Intecap-Logo](https://intecap.edu.gt/wp-content/uploads/2021/06/logo-intecap-760x336.png)
### Descripción
Esto fue un ejercicio para experimentar el uso de Media Queries en CSS.
Se probó con una estructura **Grid** y centrando con **Flex**
#### Tecnologías Utilizadas:
| HTML | CSS | 
|--|--|
| Para formar la estructura | CSS externo para diseño y sistema responsivo 

#### Características 
- 2 Paginas de ejercicios y un reto de estructura responsiva con media Queries 
- Maquetación con GRID y FLEX
- Cards para probar el sistema grid responsivo y funcional.
- Cada pagina tiene su archivo CSS dedicado a ella para mayor comprensión
